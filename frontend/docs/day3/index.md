# Day 3: AI Deployment & Spec-Driven Development

Welcome to Day 3 of the SIEHS Agentic AI Training! Today you'll deploy AI chatbots and learn strategic development approaches.

## Learning Objectives

By the end of Day 3, you will be able to:

- Build AI-powered auto email responders
- Configure OpenAI Assistants (AgentKit)
- Create and customize chat widgets (ChatKit)
- Deploy applications to Vercel
- Understand Spec-Driven Development (SDD)
- Create an AI roadmap for your department

## Modules

### Module 1: AI Auto Email Responder
- Email Trigger Setup (IMAP/Gmail)
- OpenAI Node Configuration
- Prompt Engineering for Auto-Responses
- Escalation Logic (Emergency Detection)
- **[Tutorial: AI Auto Email Responder](./tutorials/auto-email-responder)** - Build your first AI-powered automation

### Module 2: OpenAI AgentKit
- What Are AI Agents?
- OpenAI Account Setup
- Agent Builder Playground Tour
- Configure Agent Instructions & Tools
- Adding Knowledge Bases
- **[Tutorial: Weather Agent with AgentKit](./tutorials/agentkit-weather-agent)** - Build your first AI Agent

### Module 3: ChatKit Widget
- Chat UI Overview
- ChatKit Architecture
- Project Setup from Template
- Customization (Branding, Styling)
- **[Tutorial: ChatKit Widget Deployment](./tutorials/chatkit-widget)** - Connect Agent to Chat UI

### Module 4: Vercel Deployment
- GitHub Basics
- Vercel Account Setup
- Environment Variables Configuration
- Deployment Process
- **Covered in ChatKit Tutorial** - Full deployment walkthrough

### Module 5: Spec-Driven Development
- Why Specifications Matter for AI
- SDD Principles
- Writing Effective Specifications
- **Exercise**: Write SIEHS Feature Spec

### Module 6: SIEHS AI Strategy
- Three-Day Synthesis
- Prioritization Framework
- Roadmap Development
- Governance and Responsible AI
- **Project**: Departmental AI Roadmap

## Deliverables

1. **Live Deployed SIEHS Chatbot**: A functional chatbot accessible via public URL
2. **AI Roadmap**: 3-initiative AI adoption plan for your department

## Duration

**8 hours** (including breaks and exercises)
