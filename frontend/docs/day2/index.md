# Day 2: n8n Workflow Automation

Welcome to Day 2 of the SIEHS Agentic AI Training! Today you'll learn to build automated workflows without writing code.

## Learning Objectives

By the end of Day 2, you will be able to:

- Understand workflow automation concepts
- Navigate the n8n interface confidently
- Connect Google Sheets and Gmail
- Build automated email workflows
- Implement error handling and monitoring

## Modules

### Module 1: Introduction to Automation
- What is Workflow Automation?
- n8n Overview and Philosophy
- Cloud vs Self-Hosted Options
- SIEHS Automation Opportunities

### Module 2: n8n Interface Mastery
- n8n Account Setup
- Workspace Tour
- Nodes Explained
- Connections and Data Flow
- Credentials Management
- **Exercise**: First Simple Workflow

### Module 3: Google Sheets Integration
- Google Cloud Console Setup
- OAuth Configuration
- Reading Data from Sheets
- Writing and Updating Data
- **Exercise**: Sheet Reader Workflow

### Module 4: Gmail Automation
- Gmail Node Setup
- Sending Emails
- Variables and Expressions
- HTML Email Templates
- **Project**: Google Sheets → Gmail Automation

### Module 5: Production Readiness
- Error Handling with Error Trigger
- Notifications and Alerts
- Scheduling Workflows (Cron)
- Security Considerations for SIEHS
- **Exercise**: Add Error Handling

### Module 6: SIEHS Workshop
- Brainstorm Use Cases
- Design Your Next Workflow
- Implementation Planning

## Deliverable

**Working Email Automation**: A complete Google Sheets → Gmail workflow that automatically sends personalized emails.

## Duration

**8 hours** (including breaks and exercises)
