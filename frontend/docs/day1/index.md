# Day 1: Prompt Engineering Fundamentals

Welcome to Day 1 of the SIEHS Agentic AI Training! Today you'll learn how to communicate effectively with AI systems.

## Learning Objectives

By the end of Day 1, you will be able to:

- Understand what Large Language Models (LLMs) are and how they work
- Apply zero-shot and few-shot prompting techniques
- Use chain-of-thought reasoning for complex problems
- Configure AI parameters for different use cases
- Create a personal prompt library for SIEHS tasks

## Modules

### Module 1: Introduction to AI & LLMs
- What is Generative AI?
- How LLMs Work (the prediction engine)
- Prompt vs Response Cycle
- SIEHS AI Opportunity Landscape

### Module 2: Prompt Engineering Fundamentals
- Zero-Shot Prompting
- One-Shot and Few-Shot Prompting
- System Prompts and Context
- Role-Based Prompting
- **Exercise**: Emergency SMS Templates

### Module 3: Advanced Prompting Techniques
- Chain of Thought (CoT)
- Self-Consistency
- Step-Back Prompting
- ReAct Framework
- **Exercise**: Complex Decision Scenarios

### Module 4: Configuration & Best Practices
- Temperature and Creativity Control
- Token Limits and Context Windows
- Output Formatting
- Healthcare Safety Guidelines
- **Exercise**: Parameter Experimentation

### Module 5: SIEHS Applied Workshop
- Triage Communication Prompts
- Incident Report Summarization
- Patient FAQ Automation
- Training Content Generation
- **Project**: Build Your Prompt Library

## Deliverable

**Personal Prompt Library**: A collection of 5+ tested prompt templates for your SIEHS department.

## Duration

**8 hours** (including breaks and exercises)
